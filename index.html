Semantic HTML Tags

The following table describes some of the tags:

Tag Description
<header> The header tag represents introductory content in the web page like, a main heading, navigational links, a logo, etc.
<nav> nav is the navigational element, usually placed inside header or footer, and contains links to content in the site
<main> Represents the main content of the site
<section> Defines a generic section in the site
<article> Represents an article that contains information, that is independent and can be reused
<figure> May contain media like photos, images diagrams, etc. May be used with a figcaption to describe the content
<figcaption> Describes the media represented by figure
<video> Add video content to the website
<audio> Add audio content to the website
<aside> Contains information indirectly related to the main content.
<footer> The end of a block of content. may contain links, copyright information, author's information, etc.
Check Content sources for more information.
